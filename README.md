# Employee Salary & Promotion Prediction 💼

This project demonstrates the development of a multi-output machine learning system that predicts an employee's **salary** (regression) and **promotion eligibility** (classification) from their professional profile. A synthetic HR dataset was generated, containing features such as years of experience, weekly working hours, performance score, certifications, education level, city tier, job role, and company size.

The dataset was preprocessed through a machine learning pipeline that scales numerical features, one-hot encodes the nominal `role` feature, and ordinal-encodes the `edu`, `city`, and `size` features. A **multi-output deep neural network** (built with TensorFlow/Keras Functional API) was trained on a shared trunk of hidden layers that splits into two branches: a regression head predicting salary and a classification head predicting whether the employee will get a promotion.

To make the project interactive, a graphical user interface (GUI) was developed using Tkinter. The interface allows users to enter employee details and instantly receive both predictions from the trained model.

Overall, this project demonstrates a complete machine learning workflow, including data generation, preprocessing, multi-output model training, evaluation, model persistence, and deployment through a desktop application.

## 🧠 Features

- Synthetic dataset generation with realistic employee features (experience, working hours, performance score, certifications, education level, city tier, job role, company size)
- **Salary** target engineered as a weighted combination of experience, working hours, performance, certifications, education, city tier, job role, and company size, plus random noise
- **Promotion** target engineered from a logistic function of experience, performance score, certifications, and education level, thresholded at 0.5
- Preprocessing pipeline with `StandardScaler` + `SimpleImputer` (numeric), `OneHotEncoder` (nominal `role`), and `OrdinalEncoder` (ordinal `edu`, `city`, `size`) via `ColumnTransformer`
- **Multi-output neural network** built with the Keras Functional API:
  - Shared trunk: `Dense(8) → Dense(4) → Dense(16) → Dropout(0.2)`
  - Regression branch: `Dense(16)` → linear output neuron (`reg_out`) predicting salary
  - Classification branch: `Dense(8)` → sigmoid output neuron (`clf_out`) predicting promotion eligibility
  - Branches fused via `Concatenate` before the final outputs
- Compiled with the Adam optimizer, using MSE loss for the regression head and binary cross-entropy for the classification head
- Trained for 100 epochs, reaching ~78.4% promotion classification accuracy on the test set
- Trained model and preprocessing pipeline persisted with `joblib` for reuse
- Desktop GUI (Tkinter) with dropdown menus for categorical fields that loads the saved model/pipeline and returns instant salary and promotion predictions

## 🚀 Usage

1. **Train the model** (optional — pre-trained `.joblib` files are already included):
   Open and run all cells in `notebook.ipynb`. This will regenerate `model.joblib` and `pp.joblib`.

2. **Run the GUI:**
   Open `gui.ipynb` and run all cells (or convert it to a `.py` script and run it directly). A window will open where you can enter employee details and click **"Show Results"** to get the predicted salary and promotion eligibility.
